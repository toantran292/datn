# UTS Design System

## Visual Identity System

### Primary Color Palette
- **UTS Orange:** `#FF8800` - for primary actions, highlights, unread indicators
  Hover: `#FF7700`, Active: `#E56600`
- **UTS Teal:** `#00C4AB` - for focus, selection, links, badges
  Hover: `#00B9A0`, Active: `#00A896`
- **Gradient Accent:** `linear-gradient(135deg, #FF8800 0%, #00C4AB 100%)`
  Used sparingly for call-to-action buttons, loading states, or huddle status bars.

### Neutral Foundations

#### Light Mode
- Background: `#F9FAFB`
- Sidebar: `#F1F5F9`
- Surface (cards, chat bubbles): `#FFFFFF`
- Border: `#E2E8F0`
- Text: `#0F172A` primary, `#475569` secondary

#### Dark Mode
- Background: `#0B1220`
- Sidebar: `#1E293B`
- Surface: `#111827`
- Border: `#334155`
- Text: `#F9FAFB` primary, `#94A3B8` secondary

### Accents
- Success: `#22C55E`
- Warning: `#FACC15`
- Error: `#EF4444`
- Mention highlight: pale orange wash (`#FFF4E6`)
- Notification badge: vibrant pink (`#FF3B69`)

### Typography
- Typeface: **Inter**
- Headings: Semi-bold / Bold, 20-24 px
- Body: Medium 14-15 px
- Timestamp / metadata: 12 px muted
- Line height: relaxed and readable
- Tone: calm, confident, contemporary

### Shapes & Shadows
- Rounded corners: 12-16 px for cards & modals
- Soft shadow: `0 4px 16px rgba(15,23,42,0.08)`
- Focus outline: 2 px teal glow
- Transitions: 150-200 ms ease-out for hover/focus
