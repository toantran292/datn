# Meet Web

Video conferencing application built with Next.js and Jitsi.

## Features

- Real-time video/audio calls
- Screen sharing
- Live captions with speech recognition
- Virtual backgrounds
- Reactions
- Recording
- Dark/Light theme support

## Getting Started

```bash
pnpm install
pnpm dev
```

## Tech Stack

- Next.js 14
- Jitsi Meet
- Tailwind CSS
- Framer Motion
